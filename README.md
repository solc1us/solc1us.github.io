Website sederhana ini dibuat untuk menjalankan dan membandingkan algoritma Tower of Hanoi menggunakan pendekatan rekursif dan iteratif.
Aplikasi ini menampilkan jumlah langkah dan running time dari kedua algoritma sebagai bahan analisis kompleksitas algoritma.

Project ini dibuat untuk memenuhi tugas besar mata kuliah Analisis Kompleksitas Algoritma.

# Tujuan
- Mengimplementasikan Tower of Hanoi versi rekursif dan iteratif
- Membandingkan jumlah langkah dan running time
- Menghubungkan analisis teori dengan hasil implementasi

# Fitur Aplikasi
- Input jumlah piringan (N)
- Menjalankan algoritma rekursif dan iteratif
- Menampilkan:
  - Jumlah langkah
  - Running time
- Perbandingan hasil kedua algoritma

# Catatan
Meskipun secara teori kedua algoritma memiliki kompleksitas waktu yang sama, hasil running time pada implementasi nyata bisa berbeda. Hal ini dipengaruhi oleh cara program dijalankan dan lingkungan sistem, bukan karena perbedaan efisiensi algoritma secara teoritis.
